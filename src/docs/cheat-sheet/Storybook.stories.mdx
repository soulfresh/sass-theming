# Storybook Components

If you use Storybook and you want to document your theme
in your Storybook, we provide a set of
components to help you do that.

## Cheat Sheet

A good place to start with this is to just copy our
[Cheat Sheet Story](/?path=/docs/cheat-sheet-cheat-sheet--foregrounds)
into your own project. We use
MDX syntax but even if you don't, it provides a good
example of how to use the Storybook components to
document your theme.

You can copy our Cheat Sheet into your project using:

```sh
$ cp ./node_modules/@thesoulfresh/sass-theming/src/docs/cheat-sheet/Cheat.stories.mdx ./src
```

### Configure SASS

The components are setup to use your theme if it's
importable using `@import 'theme'`.

- This usually requires adding your `src` directory
  to the SASS paths option.
- If your `_theme.scss` is not available in the root,
  you can add one that imports your theme wherever
  you've defined it.

### Configure Storybook

Once SASS is configured, you will need to import our
storybook component SASS. We do that by adding the following
to our storybook `preview.js` file.

`.storybook/preview.js`

```
// Import your theme definition.
import '../src/index.scss';

// Import the sass-theming storybook component styles.
import '@thesoulfresh/sass-theming/components.scss';
```
