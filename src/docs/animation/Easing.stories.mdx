# Easing Palette

The `ease()` function provides access to your animation easing
equations. This is a Named Palette with names describing the
easing like `out-hard` or `in-out-soft`.

## Default Easing Values

The following easing values are provided out of the box.

<Story name="Easing Palette">
  <AnimationEasing />
</Story>

## ease() function

| Param | Type | Description |
| --- | --- | --- |
| `name` | `key` | A key in the `$palette-ease` map. |
| __Return__ | `cubic-bezier` | A value that can be used as an easing equation. |

```scss
.foo {
  transition: margin-right speed(m) ease(out-hard);
}

.bar {
  animation: some-anim speed(l) ease(in-out-bounce);
}
```

## Configuration

The easing palettes are available through the `ease`
property of each theme.
