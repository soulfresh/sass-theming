# Shadows Palette

This is a Named Palette used to access box shadows behind/around/inside elements.
There are only a couple shadows provided out of the box as shadow usage varies
greatly by project.

## `shadow()` function

| Param | Type | Default | Description |
| --- | --- | --- | --- |
| `name` | `key` | `overlay` | A key in the `$palette-shadow` or `$palette-shadow-dark` palette map. |
| __Return__ | `box-shadow` | | A CSS box-shadow definition |

```css
.foo {
  box-shadow: shadow(overlay);
}
```

## Configuration

The shadow palettes are available through the `shadow`
property of each theme.
