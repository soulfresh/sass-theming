# Gradients Palette

This Named Palette can be used to add gradients to an element backgroud.

## `gradient()` function

| Param | Type | Default | Description |
| --- | --- | --- | --- |
| `name` | `key` | `light` | A key in the `$palette-shadow` or `$palette-shadow-dark` palette map. |
| __Return__ | `box-shadow` | | A CSS box-shadow definition |

```css
.foo {
  background: linear-gradient( #{color(gradients, active)} );
}
```

## Configuration

The gradient palettes are available through the `gradient`
property of each theme.
