# Font Families & Sizes

The `font-family()` function provides a Named Palette of
font family definitions named by their usage (ex. `title`, `body`, etc).

## Default Font Families

The following default font families are configured out of the box.

  <FontFamilies />
</Story>

## `font-family()` function

| Param | Type | Default | Description |
| --- | --- | --- | --- |
| `name` | `key` | `body` | A key in the `$palette-font-family` palette map. |
| __Return__ | `list` | | A CSS font-family list |

```css
.foo {
 font-family: font-family(title);
}

.bar {
  font-family: font-family(code);
}
```

## Global Variables

In addition to the `font-family()` function, there are a couple
font family global variables that are useful in generating font
definitions. You can use them as your fallbacks after your desired font
face.

- `$sans-serif`: Fallbacks to for your Sans Serf faces.
- `$monospace`: Fallback to use for your Code font.

Here's how you can use them:

```css
$my-title-font: Roboto $sans-serif;
$my-code-font: Menlo $monospace;
```

## Configuration

To configure the font families for your project, provide or modify
the `$palette-font-family` map.

```css
$palette-font-family : (
  body  : (Roboto, $sans-serif),
  title : ("HelveticaNeue-Light", "Helvetica Neue Light", $sans-serif),
  code  : (Monaco, $monospace),
);

@import '@thesoulfresh/sass-theming/theme';
```
