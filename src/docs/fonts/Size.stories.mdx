# Font Sizes

The `font-size()` function provides access to a Named Palette
of font sizes using s/m/l naming.

## Default Font Sizes

The following sizes are provided out of the box.

<Story name="Font Size Palette">
  <FontSizes />
</Story>

## `font-size()` function

| Param | Type | Default | Description |
| --- | --- | --- | --- |
| `name` | `key` | `m` | A key in the `$palette-font-size` palette map. |
| __Return__ | `px number` | | A CSS pixel value |

```css
.foo {
  font-size: font-size(s);
}

.bar {
  font-size: font-size(xxl);
}
```

## Configuration

To configure the font sizes for your project, provide or modify
the `$palette-font-size` map.

```css
$palette-font-size: (
  xxxl    : 36,
  xxl     : 32,
  xl      : 24,
  l       : 16,
  m       : 14,
  s       : 12,
  xs      : 10
);

@import '@thesoulfresh/sass-theming/theme';
```
